# Mobile Face Detection App

📱 Academic Project  
🎓 Grade: 91  

## Description
Developed an Android mobile application for face detection and recognition in real-time.

## Features
- Real-time face detection using camera
- Face recognition with confidence score
- Firebase Firestore integration for storing data
- User-friendly mobile interface

## Technologies
- Android (Java/Kotlin)
- Firebase Firestore
- Machine Learning (Face Recognition)

## Notes
Developed as part of a team project.
